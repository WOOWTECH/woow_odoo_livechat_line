LiveChat LINE Integration

Seamlessly connect LINE Messaging API with Odoo LiveChat

v18.0.1.0.0 | by WoowTech

Enable your customer service team to handle LINE conversations directly inside the standard Odoo Discuss interface. Incoming LINE messages automatically create LiveChat sessions, and agent replies are delivered back to the customer on LINE — all in real time.

Key Features

⚙

LINE Channel Configuration

Configure Channel ID, Channel Secret, and Channel Access Token directly from the LiveChat channel form. Per-channel settings enable multi-tenant deployments.

🔒

Webhook with HMAC-SHA256 Verification

Every incoming webhook request is validated using HMAC-SHA256 signature verification against your Channel Secret, ensuring only authentic LINE platform events are processed.

👥

Automatic Guest & Partner Creation

First-time LINE users are automatically registered as mail.guest records. The LINE Profile API populates display name and avatar. Guests can later be promoted to full res.partner contacts.

👤

LINE Profile API Integration

Fetches the user's LINE display name and profile picture automatically when a new conversation is created, giving agents immediate context about the customer.

💬

Rich Message Support

Handles text, image, video, audio, file, sticker, and location messages from LINE, converting each type into the appropriate Odoo attachment or message body.

⇄

Bidirectional Messaging

Messages flow both ways: LINE→Odoo via webhook, and Odoo→LINE via the Push Message API. Agents reply from Discuss and customers receive responses instantly on LINE.

🔗

Guest-to-Partner Linking Wizard

A dedicated wizard lets agents link an anonymous LINE guest to an existing or new res.partner record, preserving the full conversation history under the contact.

🏗

Multi-tenant Support

Each LiveChat channel can be connected to a different LINE Official Account. Webhook routing automatically dispatches events to the correct channel based on the destination user ID.

Quick Setup

  1. Create a LINE Official Account and enable the Messaging API in the LINE Developers Console.
  2. Install this module in your Odoo 18 instance.
  3. Open a LiveChat channel and fill in the LINE tab: Channel ID, Channel Secret, and Channel Access Token.
  4. Set the Webhook URL in the LINE Developers Console to:
    https://your-odoo.com/line/webhook/<livechat_channel_id>
  5. Send a test message from LINE — it should appear in Discuss.
